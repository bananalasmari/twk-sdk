Two Ways to Access APIs

The SDK provides two flexible ways to access its functionality:

Method 1: Direct Access (Convenience)

Use direct methods for quick access to commonly used functions:

// Direct access to common methods
const deviceInfo = await TWKSDK.getDeviceInfo();
const appearance = await TWKSDK.getAppearance();
const userName = await TWKSDK.getUserFullName();

Method 2: Organized by Category

Use categorized namespaces for organized access to all features:

// Organized by category
const deviceInfo = await TWKSDK.Device.getInfo();
const appearance = await TWKSDK.Device.getAppearance();
const userName = await TWKSDK.User.getFullName();

// Access to all specialized features
const photo = await TWKSDK.Camera.takePhoto();
const location = await TWKSDK.Permissions.requestLocation();
const vehicles = await TWKSDK.Vehicles.getList();

Device APIs

Access device information including appearance mode, language, and system capabilities.

TWKSDK.Device.getInfo() or TWKSDK.getDeviceInfo()

Get comprehensive device information including appearance mode and language.

// Organized access
const info = await TWKSDK.Device.getInfo();

// Or direct access
const info = await TWKSDK.getDeviceInfo();

console.log(`Appearance: ${info.result.appearance}, Language: ${info.result.language}`);

TWKSDK.Device.getAppearance() or TWKSDK.getAppearance()

Get the device appearance mode. Returns 1 for light mode, 2 for dark mode.

// Organized access
const appearance = await TWKSDK.Device.getAppearance();

// Or direct access (convenience)
const appearance = await TWKSDK.getAppearance();

const isDark = appearance === 2;

TWKSDK.Device.isDarkMode()

Check if device is in dark mode.

const isDark = await TWKSDK.Device.isDarkMode();
if (isDark) {
  // Apply dark theme
}

TWKSDK.Device.getLanguage()

Get the device language preference (e.g., 'en', 'ar').

const language = await TWKSDK.Device.getLanguage();

User APIs

Access user profile data and personal information.

TWKSDK.User.getFullName() or TWKSDK.getUserFullName()

Get the user's full name (automatically uses V2 endpoint when available).

// Organized access
const name = await TWKSDK.User.getFullName();

// Or direct access (convenience)
const name = await TWKSDK.getUserFullName();

if (name) {
  console.log(`Welcome, ${name}!`);
}

Available User Methods

  • getId() - Get user's unique identifier
  • getType() - Get user's account type
  • getBirthDate() - Get user's birth date
  • getMobileNumber() - Get user's mobile number
  • getGender() - Get user's gender
  • getLocation() - Get user's current location coordinates
  • getNationality() - Get user's nationality name
  • getNationalityISO() - Get user's nationality ISO code
  • getMaritalStatus() - Get user's marital status
  • getHealthStatus() - Get user's health status
  • getDisabilityType() - Get user's disability type (if applicable)
  • getBloodType() - Get user's blood type
  • getNationalAddress() - Get user's national address
  • getDegreeType() - Get user's educational degree type
  • getOccupation() - Get user's occupation
  • getProfilePhoto() - Get user's profile photo
  • getImage(nationalId) - Get image for a specific national ID

Family APIs

Access family members and sponsored individuals with optional filtering.

TWKSDK.Family.getMembers(options?)

Get list of family members with optional age and gender filters.

// Get all family members
const allMembers = await TWKSDK.Family.getMembers();

// Get children between 5-18
const children = await TWKSDK.Family.getMembers({ minAge: 5, maxAge: 18 });

// Get female family members
const females = await TWKSDK.Family.getMembers({ gender: 'female' });

TWKSDK.Family.getSponsors(options?)

Get list of sponsored individuals with optional filters.

const sponsors = await TWKSDK.Family.getSponsors({ minAge: 18, maxAge: 60 });

Vehicles APIs

Manage vehicles and traffic violations.

TWKSDK.Vehicles.getList()

Get list of vehicles registered to the user.

const vehicles = await TWKSDK.Vehicles.getList();

TWKSDK.Vehicles.getAllViolations()

Get all violations (both paid and unpaid).

const { paid, unpaid } = await TWKSDK.Vehicles.getAllViolations();
console.log(`Unpaid violations: ${unpaid.result.length}`);
  • getUnpaidViolations() - Get unpaid traffic violations
  • getPaidViolations() - Get paid traffic violations

Gallery APIs

Access photos and videos from the device gallery.

TWKSDK.Gallery.pickImage()

Pick a single image from the gallery.

const image = await TWKSDK.Gallery.pickImage();
if (image.success) {
  console.log('Image selected:', image.result);
}
  • pickImages() - Pick multiple images
  • pickVideo() - Pick a single video
  • pickVideos() - Pick multiple videos
  • getRawData(fileName) - Get raw file data

Camera APIs

Capture photos and videos using the device camera.

TWKSDK.Camera.takePhoto()

Take a photo using the device camera.

const photo = await TWKSDK.Camera.takePhoto();
if (photo.success) {
  console.log('Photo captured:', photo.result);
}

TWKSDK.Camera.recordVideo()

Record a video using the device camera.

const video = await TWKSDK.Camera.recordVideo();

Files APIs

Pick and manage files with base64 encoding support.

TWKSDK.Files.pickFileBase64()

Pick a file and get its content as base64.

const file = await TWKSDK.Files.pickFileBase64();
if (file.success) {
  const { mime_type, data } = file.result;
  // Use the base64 data
}

TWKSDK.Files.pickFile()

Pick a file and get its ID.

const file = await TWKSDK.Files.pickFile();

Permissions APIs

Request device permissions from the user.

TWKSDK.Permissions.requestAll()

Request all common permissions at once.

const perms = await TWKSDK.Permissions.requestAll();
console.log('Location:', perms.location.success);
console.log('Camera:', perms.camera.success);
console.log('Gallery:', perms.gallery.success);
console.log('Notifications:', perms.pushNotification.success);

Individual Permission Methods

  • requestLocation() - Request location permission
  • requestPreciseLocation() - Request precise location permission
  • requestCamera() - Request camera permission
  • requestGallery() - Request gallery permission
  • requestPushNotification() - Request push notification permission

Authentication APIs

Authentication and security features.

TWKSDK.Auth.authenticateBiometric()

Authenticate user using biometric (fingerprint/face recognition).

const result = await TWKSDK.Auth.authenticateBiometric();
if (result.success) {
  console.log('Biometric authentication successful');
}

TWKSDK.Auth.generateToken()

Generate an authentication token (uses V2 endpoint when available).

const token = await TWKSDK.Auth.generateToken();

Share APIs

Share content and screenshots.

TWKSDK.Share.screenshot()

Share a screenshot of the current screen.

await TWKSDK.Share.screenshot();

TWKSDK.Share.base64Content()

Share base64-encoded content with specified mime type.

await TWKSDK.Share.base64Content(
  'document.pdf',
  base64Data,
  'application/pdf'
);

Navigation APIs

Navigate to different screens and services in the TWK app.

TWKSDK.Navigation.openScreen()

Open a specific screen in the TWK app.

await TWKSDK.Navigation.openScreen('profile', { userId: '123' });

TWKSDK.Navigation.openService()

Open a specific service by its ID.

await TWKSDK.Navigation.openService('service-id', { param: 'value' });

Scanner APIs

Scan QR codes and barcodes.

TWKSDK.Scanner.scanCode()

Scan a QR code or barcode.

const result = await TWKSDK.Scanner.scanCode();
if (result.success) {
  console.log('Scanned:', result.result);
}

Cards APIs

Manage TWK cards with CRUD operations.

TWKSDK.Cards.create()

Create a new card.

await TWKSDK.Cards.create({
  title: 'New Card',
  description: 'Card description',
  data: { key: 'value' }
});
  • post(actionType, payload) - Post a card action
  • update(payload) - Update an existing card
  • delete(payload) - Delete a card

Payment APIs

Configure payment callback URLs and page names.

TWKSDK.Payment.configure()

Configure payment callback URLs and page names.

TWKSDK.Payment.configure({
  callbackSuccessUrlList: ['https://example.com/success'],
  callbackFailureUrlList: ['https://example.com/failure'],
  successPageName: 'Payment Success',
  failurePageName: 'Payment Failed'
});

Utility Methods

Additional helper methods for SDK management.

TWKSDK.isAvailable()

Check if the TWK bridge is available.

if (TWKSDK.isAvailable()) {
  // TWK bridge is ready
}

TWKSDK.getVersion()

Get the current SDK version.

const version = TWKSDK.getVersion();
console.log('SDK Version:', version);

TWKSDK.getAvailableMethods()

Get information about available TWK methods.

const methods = TWKSDK.getAvailableMethods();
console.log('V1 Methods:', methods.v1);
console.log('V2 Methods:', methods.v2);

Convenience Methods Summary

The SDK provides direct access methods for commonly used functions. These are shortcuts to the most frequently accessed APIs.

Available Convenience Methods

  • TWKSDK.getDeviceInfo() - Direct access to device information (alias for Device.getInfo())
  • TWKSDK.getAppearance() - Direct access to appearance mode (alias for Device.getAppearance())
  • TWKSDK.getUserFullName() - Direct access to user's full name (alias for User.getFullName())

Quick Start Example

Use convenience methods for rapid development:

// Initialize SDK
TWKSDK.init({
  apiBase: 'https://api.example.com',
  enableLog: true
});

// Quick access to common features
const userName = await TWKSDK.getUserFullName();
const isDark = await TWKSDK.getAppearance() === 2;

console.log(`Hello ${userName}!`);
console.log(`Dark mode: ${isDark}`);
